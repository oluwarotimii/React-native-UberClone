## Dependencies
	

*  Vector Icons,
 * Google Maps Api
*  Gesture Handler,
 * Paystack Webview,
*  SafeArea Context,
*  React-Navigation,
* React-Redux
* Redux ToolKit
* Google Maps Direction
*   React-native google maps Autocomplete
* TailWind






## Run Locally

Clone the project

```bash

  git clone. https://github.com/oluwarotimii/React-native-UberClone.git

```

Go to the project directory

```bash

  cd React-native-UberClone

```

Install dependencies

```bash

  npm install

```

Start the server

```bash

  npm start

```
